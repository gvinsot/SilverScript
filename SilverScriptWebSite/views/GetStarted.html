SilverScript is a javascript framework built to bring advanced Xaml binding syntaxes to javascript and typescript.




Discover the easiest way to develop a website extensively using MVVM development pattern. SilverScript is designed to reduce significantly the need of creating code by connecting your interface direclty to data sources and bind directly your UI elements.

We support:
    - OneTime, OneWay and TwoWay binding modes.
    - Data Templates for items lists
    - Converters
    - Multiple Bindings
    - Binding to json sources
 
We had special attention to avoid memory leaks and performance bottlenecks. XAML developers will find it easy to learn using a familiar syntax.

Set the datacontext
<div data-context="/api/mycontroller">...</div>

Add your bindings
<Button data-binding="{Binding Path=name,Destination=Content}"> </Button>

Bind a different property
<a href="blank" data-binding="{Binding Path=name,Destination=href}"> My dynamic link </a>

Mix syntaxes
<div data-context="/viewmodels/mycontroller">
    <button data-binding="{Binding Path=name,Destination=Content}" data-context="/mycontroller/{DataBinding Path=id}"></button>
</div>
