Test Page



Apply a template from a json source sending 1000 values (250 lines).
Sample data model (C# model that is serialized to json through .Net WebApi):

public class TestClass
{
public string FirstName = "FirstName_" + Path.GetRandomFileName();
public string LastName = "LastName_"+ Path.GetRandomFileName();
public string Address = (new Random(DateTime.Now.TimeOfDay.Milliseconds)).Next(200)+"th " + Path.GetRandomFileName() + " street";
public int Age = (new Random(DateTime.Now.TimeOfDay.Milliseconds)).Next(100);
}


The view :
<table style="width:100%" data-source="/viewmodels/Test" data-template="/views/item.html">
</table>

And template (item.html):

<tr>
<td data-binding="{Binding Path=FirstName,Destination=Content}"></td>
<td data-binding="{Binding Path=LastName,Destination=Content}"></td>
<td data-binding="{Binding Path=Address,Destination=Content}"></td>
<td data-binding="{Binding Path=Age,Destination=Content}"></td>
 </tr>
